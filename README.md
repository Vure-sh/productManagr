# 🛒 Product Manager

A simple product management web application built with **HTML, CSS, and JavaScript**. The project allows users to create, edit, delete, and persist products using the browser's Local Storage.

## Features

* ➕ Add new products
* ✏️ Edit existing products
* 🗑️ Delete products
* 💾 Automatic data persistence with Local Storage
* 🔄 Data remains available after refreshing the page
* 📱 Clean and responsive user interface

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Local Storage API

## Project Structure

├── index.html
├── style.css
├── script.js
└── README.md

## How It Works

1. Enter a product name, price, and quantity.
2. Click **Save** to add the product.
3. Use the **Edit** button to modify a product.
4. Use the **Delete** button to remove a product.
5. All changes are automatically saved in Local Storage.
